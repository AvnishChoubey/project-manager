# 📌 Project Management Application – Backend

A robust and scalable backend service for a Project Management Application built using **Spring Boot**.  
This API provides endpoints to manage projects, tasks, users, and related workflows.

---

## 🚀 Tech Stack

- **Java 21+**
- **Spring Boot**
- **Spring Web**
- **Spring Data JPA**
- **Spring Security**
- **Hibernate**
- **Maven**
- **MySQL**
- **JWT Authentication**

---

## ✨ Features

- 👤 User management (Create, update, delete, assign roles)
- 📁 Project management
- ✅ Task management
- 📅 Deadline tracking
- 🔐 Authentication & Authorization (JWT-based)
- 📊 RESTful API design
- 🧾 Exception handling & validation
- 📦 Layered architecture (Controller → Service → Repository)
